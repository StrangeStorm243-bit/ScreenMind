- **No previous projects** — must be built from scratch during the event
- **GitHub repo required** — public, submitted via Devpost
- **3-minute demo recording** required with Devpost submission
- Must use **at least 3 sponsor tools**
- Late submissions are cut off — no exceptions
- Government-issued photo ID required
- Must be 18+
- Follows AWS Community Code of Conduct and Event Terms
- **Entry is first come, first serve** — even with registration + Luma QR code, venue capacity is hard-capped
- Must have **Luma QR code/ticket** to enter (enter via Market Street)
- No e-bikes/scooters in the venue
- Student IDs NOT accepted — must be government-issued (driver's license, passport, etc.)

- **Demo matters**: 20% of score — rehearse, keep it tight, show the agent working live
